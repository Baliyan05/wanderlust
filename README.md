# рџ“– About

**Wanderlust** is a full-stack Airbnb-inspired web application developed using **Node.js, Express.js and MongoDB Atlas**.

Users can browse vacation rentals, create their own listings, upload property images, manage reviews, and securely authenticate using Passport.js.

The application follows the **MVC Architecture**, implements **RESTful APIs**, and is deployed on **Render** with **MongoDB Atlas** as the cloud database.
